# 衛生管理者試験 対策サイト

第一種・第二種衛生管理者試験の過去問＆オリジナル想定問題を収録した無料学習サイトです。

## 収録内容

| 種別 | モード | 分野 | 問題数 |
|------|--------|------|--------|
| 第一種 | 過去問（令和2〜6年度） | 関係法令 17問 × 5年 | 85問 |
| | | 労働衛生 17問 × 5年 | 85問 |
| | | 労働生理 10問 × 5年 | 50問 |
| | オリジナル想定問題 | 各分野 20問 × 3分野 | 60問 |
| 第二種 | 過去問（令和2〜6年度） | 各分野 10問 × 5年 × 3分野 | 150問 |
| | オリジナル想定問題 | 各分野 20問 × 3分野 | 60問 |
| **合計** | | | **490問** |

## 技術構成

- HTML / CSS / JavaScript（フレームワーク不使用）
- PWA対応（オフライン利用・ホーム画面追加可能）
- 全53ページの静的サイト（GitHub Pages でホスティング）

## サイト構造

```
/                              トップページ
/type1/                        第一種トップ
/type1/past/                   過去問 年度一覧
/type1/past/2024/              令和6年度 分野一覧
/type1/past/2024/law/          関係法令 問題ページ
/type1/original/               想定問題 分野一覧
/type1/original/law/           関係法令 問題ページ
/type2/...                     第二種（同構造）
```

## ローカルで動作確認する方法

```bash
# 任意のローカルサーバーで起動（例：Python）
python -m http.server 8000

# ブラウザで http://localhost:8000 を開く
```

## コンテンツの更新方法

1. `js/data.js` の問題データを編集
2. 新しい年度・分野のHTMLページが必要な場合は追加
3. `sw.js` の `CACHE_NAME` のバージョン番号を上げる
4. `git push` でデプロイ完了
