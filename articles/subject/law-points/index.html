関係法令の出題傾向まとめ
【第一種・第二種別】

衛生管理者試験の「関係法令」は、3科目の中で最も問題数が多く、合否を大きく左右する科目です。第一種では全44問中17問、第二種でも全30問中10問が関係法令から出題されます。法律の条文に基づく問題が中心のため、「何の数字を覚えれば得点できるか」を把握した上で戦略的に学習することが合格への近道です。

この記事では第一種・第二種それぞれの出題構成と頻出テーマを体系的に解説し、効率的な攻略法まで詳しくお伝えします。

1. 関係法令とはどんな科目か - 構成と配点

関係法令は、労働安全衛生法をはじめとする各種法律・規則の内容を問う科目です。試験では条文の内容を正確に理解しているかどうかが問われ、特に具体的な数値（選任人数・実施頻度・保存期間など）を問う問題が多く出題されます。

試験における位置づけと出題数

第一種衛生管理者第二種衛生管理者
関係法令の総問題数 17問 10問
　うち有害業務以外 10問 10問（全問）
　うち有害業務関連 7問 なし
試験全体の問題数 44問 30問
関係法令の割合 約39% 約33%
科目合格基準 40%以上（7問以上） 40%以上（4問以上）

第一種では17問と全科目中最多の出題数を誇り、試験の得点を最も左右する科目です。第二種は10問ですが、全30問のうちの3分の1を占めるため、油断できません。どちらの試験でも関係法令で高得点を取ることが合格の最重要条件といえます。

関係法令の主な根拠法令

試験に出題される関係法令は複数の法律・規則にまたがっています。出題の大半は以下の法令から出ます。

労働安全衛生法・同施行令・規則
出題の約60〜70%
衛生管理体制・健康診断・安全衛生教育など試験の中核をなす法令
労働基準法・同施行規則
出題の約20〜25%
労働時間・休憩・休日・年次有給休暇・女性保護など
有害業務関連の特別規則（第一種のみ）
第一種の約30%
特定化学物質障害予防規則・有機溶剤中毒予防規則・電離放射線障害防止規則・粉じん障害防止規則など
関係法令の学習で最重要な心構え
関係法令は「法律の条文を丸暗記する科目」ではありません。条文の趣旨（なぜその規定があるのか）を理解した上で、重要な数値・期間・人数を正確に覚えることが求められます。特に選択肢の作り方として「数字だけを変えた」誤り選択肢が多用されるため、数値の正確な記憶が合否を直接左右します。

2. 第二種の出題傾向 - 有害業務以外の関係法令10問を攻略すべし

第二種衛生管理者の関係法令は、有害業務に関する内容を含まない10問構成です。オフィス系・サービス系など有害業務のない職場を対象とした規定が中心で、比較的馴染みやすい内容が多いのが特徴です。

第二種 関係法令の頻出テーマ一覧

テーマ主な出題内容毎年の出題
衛生管理者の選任 労働者数と選任人数の対応・選任期限・報告義務 ほぼ毎年
衛生委員会 設置義務・構成メンバー・開催頻度・調査審議事項 ほぼ毎年
一般健康診断 実施頻度・項目・結果の記録・事後措置・報告義務 ほぼ毎年
労働時間・休憩・休日 法定労働時間・休憩の付与・週休制・変形労働時間制 毎年
年次有給休暇 付与日数・比例付与・時季指定・計画的付与 高頻度
女性・年少者保護 危険有害業務の就業制限・深夜業・坑内労働の禁止 高頻度
産業医 選任義務・職務内容・権限・作業場巡視 高頻度

第二種の10問のうち、衛生管理体制（選任・委員会）・健康診断・労働時間の3テーマだけで例年5〜6問が出題されます。まずこの3テーマを完成させることが第二種合格の最短ルートです。

第二種 関係法令 戦略的な学習優先順位
  • 最優先（毎年必出）：衛生管理者の選任 ／ 衛生委員会 ／ 一般健康診断 ／ 法定労働時間・休憩
  • 次に優先（高頻度）：産業医 ／ 年次有給休暇 ／ 女性・年少者保護
  • 余裕があれば：安全衛生教育 ／ ストレスチェック ／ 事務所衛生基準
10問中7問以上の正答（70%）を目標に設定し、最優先テーマを完璧に仕上げてから次に進む学習順序がおすすめです。

3. 第一種の出題傾向 - 有害業務関連の上乗せ7問を攻略すべし

第一種衛生管理者の関係法令は、第二種の10問（有害業務以外）に加えて、有害業務に関連する7問が上乗せされた計17問構成です。この有害業務関連の7問が第一種特有の難関であり、ここの対策が合否を分ける大きなポイントになります。

有害業務関連7問の主な出題テーマ

テーマ主な根拠法令主な出題内容
特定化学物質 特定化学物質障害予防規則（特化則） 第1類・第2類・第3類の区分、製造許可・使用禁止物質、作業主任者の選任、特殊健診の頻度
有機溶剤 有機溶剤中毒予防規則（有機則） 第1種・第2種・第3種の区分、屋内作業での規制、作業主任者、有機則健診
電離放射線 電離放射線障害防止規則（電離則） 管理区域の設定、線量限度、放射線業務従事者の健診、エックス線作業主任者
粉じん 粉じん障害防止規則（粉じん則） 特定粉じん作業・粉じん作業の区分、防じんマスクの使用義務、じん肺健診
鉛 鉛中毒予防規則（鉛則） 鉛業務の種類、作業主任者、鉛健診
高気圧作業 高気圧作業安全衛生規則 作業主任者の選任、健康診断、減圧の方法
酸素欠乏 酸素欠乏症等防止規則 第1種・第2種酸欠危険場所の区分、作業主任者、空気中の酸素濃度基準

有害業務関連7問の中で特に出題頻度が高いのは特定化学物質・有機溶剤・電離放射線の3テーマです。この3テーマだけで毎年4〜5問が出題されることも珍しくありません。

第一種・第二種の関係法令 学習時間配分の目安

パート対象試験問題数推奨学習時間
有害業務以外の関係法令 第一種・第二種共通 10問 約20〜25時間
有害業務関連の関係法令 第一種のみ 7問 約12〜15時間
有害業務関連は「名称と区分の対応」が最大の難関
特化則・有機則・粉じん則などは、物質や作業の「第1類/第2類」「第1種/第2種」といった区分とその規制内容の対応が試験の中心です。たとえば特定化学物質の第1類は製造に厚生労働大臣の許可が必要、第2類は特殊健診が6ヶ月ごとに必要、といった細かい違いを覚えることが求められます。区分とセットで規制内容を整理した一覧表を作ると効果的です。

4. 頻出テーマ① 衛生管理体制（選任義務・衛生委員会）

出題頻度：毎年必出

衛生管理体制は関係法令で最も出題頻度が高いテーマです。「衛生管理者の選任義務」「産業医の選任義務」「衛生委員会の設置と運営」の3点を正確に覚えることが合格への第一歩です。

衛生管理者の選任義務

常時50人以上の労働者を使用する事業場は、衛生管理者を選任しなければなりません。労働者数に応じた選任人数は試験頻出の数値です。

常時使用する労働者数衛生管理者の選任人数
50人以上200人以下1人以上
200人超～500人以下2人以上
500人超～1,000人以下3人以上
1,000人超～2,000人以下4人以上
2,000人超～3,000人以下5人以上
3,000人超6人以上
✓
選任すべき事由が発生した日から14日以内に選任しなければならない
✓
選任後は遅滞なく所轄の労働基準監督署長へ報告する義務がある
✓
常時1,000人超の事業場（一定の有害業務は500人超）は専任の衛生管理者を1人以上置かなければならない
✓
衛生管理者は少なくとも毎週1回、作業場等を巡視しなければならない
✓
第一種免許は全業種で選任可能。第二種免許は有害業務のない業種（情報サービス・金融・卸売・小売など）に限定

産業医の選任義務

✓
常時50人以上の事業場：産業医を1人以上選任する義務あり
✓
常時3,000人超の事業場：産業医を2人以上選任する義務あり
✓
常時1,000人以上の事業場または一定の有害業務に常時500人以上従事させる事業場では専属の産業医が必要
✓
産業医は少なくとも毎月1回（産業医が提供する情報等を活用しての場合は2ヶ月に1回）作業場を巡視する
✓
産業医は労働者の健康管理に関して勧告・指導・助言を行う権限を持つ

衛生委員会の設置と運営

✓
常時50人以上の事業場に設置義務がある
✓
毎月1回以上開催しなければならない
✓
議長は総括安全衛生管理者または事業の実施を統括管理する者が務める
✓
委員には産業医・衛生管理者・衛生に関し経験を有する労働者が含まれる
✓
委員の半数は過半数労働組合（または過半数代表者）が推薦した者でなければならない
✓
議事録は3年間保存し、労働者に周知しなければならない
「安全委員会」と「衛生委員会」の混同に注意
常時50人以上かつ一定の業種では「安全委員会」の設置も義務付けられています。安全委員会と衛生委員会を統合した「安全衛生委員会」を設置することも認められています。試験では「衛生委員会のみの設置が義務付けられる業種・規模」について問われることがあります。製造業・建設業など一定の業種は安全委員会も必要になる点を押さえておきましょう。

5. 頻出テーマ② 健康診断（種類・頻度・事後措置）

出題頻度：毎年必出

健康診断は、種類・実施頻度・記録の保存期間・事後措置・報告義務の各ポイントが毎年出題される超頻出テーマです。種類ごとに規定が異なるため、混同しないように整理することが重要です。

健康診断の種類と実施頻度

種類対象者実施頻度記録保存
雇入れ時健康診断 常時使用する労働者（雇入れ時） 雇入れの際 5年
定期健康診断 常時使用する労働者 1年以内ごとに1回 5年
特定業務従事者健康診断 深夜業など特定業務従事者 6ヶ月以内ごとに1回 5年
海外派遣労働者健康診断 6ヶ月以上海外派遣する労働者 派遣前・帰国後 5年
給食従業員健康診断 事業場附属の食堂・炊事場の従業員 雇入れ・配置換えの際 5年
特殊健康診断（有害業務） 有害業務従事者（特化則・有機則など） 6ヶ月以内ごとに1回 5年（一部30年）
✓
定期健康診断の結果は、常時50人以上の事業場では所轄労働基準監督署長に報告しなければならない
✓
雇入れ時健康診断では医師による健診結果の証明書を提出させることで省略できる項目がある
✓
パートタイム労働者でも、週の所定労働時間が通常の労働者の4分の3以上であれば定期健康診断の対象

健康診断の事後措置

健康診断後の事業者の義務も重要な出題テーマです。結果を受けて事業者がとるべき措置の流れを把握しておきましょう。

✓
健康診断の結果は本人に通知しなければならない
✓
異常の所見がある労働者について、3ヶ月以内に産業医等の意見を聴かなければならない
✓
医師の意見を勘案し、必要な場合は就業場所の変更・作業の転換・労働時間の短縮などの措置を講じる
✓
措置の内容は衛生委員会に報告するとともに、健康診断個人票に医師の意見を記載する
特殊健康診断の記録保存期間「30年」に注意
特殊健康診断の記録保存期間は原則5年ですが、一部の有害物質（特定化学物質の第1類・第2類の一部、電離放射線など）については30年間の保存が義務付けられています。「5年か30年か」という選択肢で問われるため、30年保存が必要な物質・業務の種類を別途確認しておきましょう。

6. 頻出テーマ③ 労働時間・休憩・休日

出題頻度：毎年必出

労働時間・休憩・休日に関する規定は労働基準法から出題されます。「法定労働時間の原則」「休憩の付与方法」「休日の与え方」「時間外・休日労働の36協定」「年次有給休暇」の各ポイントが毎年問われます。

法定労働時間・休憩・休日の基本

✓
法定労働時間：1日8時間・週40時間が原則（特例事業場は週44時間）
✓
休憩時間：労働時間が6時間超→45分以上、8時間超→60分以上付与する義務
✓
休憩は労働時間の途中に与えなければならない（前後に置くことは不可）
✓
休憩は一斉に与えることが原則（労使協定で例外あり）
✓
法定休日：毎週1日または4週4日の休日を与えなければならない

変形労働時間制・裁量労働制

変形労働時間制・裁量労働制は試験でよく出るテーマです。種類と適用条件を混同しないよう整理しておきましょう。

制度単位期間要件上限
1ヶ月単位の変形労働時間制 1ヶ月以内 労使協定または就業規則等 週平均40時間
1年単位の変形労働時間制 1ヶ月超〜1年以内 労使協定（届出必要） 週平均40時間・1日10時間・週52時間
フレックスタイム制 清算期間（最大3ヶ月） 就業規則＋労使協定 清算期間を平均して週40時間
専門業務型裁量労働制 — 労使協定（19業務が対象） みなし労働時間を協定で定める

年次有給休暇

✓
雇入れの日から6ヶ月間継続勤務し、全労働日の8割以上出勤した場合に10日の有給休暇が付与される
✓
最大付与日数は勤続6年6ヶ月以上で20日
✓
年10日以上の有給が付与された労働者に対して、事業者は年5日の時季指定が義務付けられている（2019年改正）
✓
有給休暇の時効は2年（付与日から2年で消滅）
✓
週所定労働日数が少ない労働者（パート等）には比例付与の規定がある
労働時間の「特例事業場（週44時間）」も押さえておく
商業・映画・演劇業・保健衛生業・接客娯楽業などの特例事業場では、常時10人未満の場合に限り週44時間の労働時間が認められています。「週40時間か44時間か」という数値の選択肢として出題されるため、特例事業場の業種リストを把握しておくと安心です。

7. 頻出テーマ④ 有害業務関連の法令（特化則・有機則・電離則）

第一種のみ出題

有害業務関連の法令は第一種固有の出題範囲であり、7問のうち毎年4〜5問がこのテーマから出ます。規制の内容が物質・業務ごとに細かく異なるため、整理して覚えることが求められます。

特定化学物質障害予防規則（特化則）

特化則は、がん・重篤な慢性疾患を引き起こすおそれがある特定の化学物質を取り扱う業務について、製造・使用・健康診断・作業環境管理を規制するものです。

区分特徴主な物質例主な規制
第1類 製造等に厚生労働大臣の許可が必要 ベンゾトリクロリド・ジクロルベンジジンなど 製造許可・特殊健診（6ヶ月ごと）・記録30年保存
第2類 取扱い規制がある（許可不要） 塩素・クロム酸・シアン化水素など多数 作業環境測定・特殊健診（6ヶ月ごと）・作業主任者
第3類 漏洩・拡散防止が中心 アンモニア・塩酸・硫酸など 設備の密閉化・漏洩防止措置

有機溶剤中毒予防規則（有機則）

有機則は、有機溶剤を取り扱う業務での健康障害を防ぐための規則です。有機溶剤は第1種・第2種・第3種に分類され、許容消費量・換気の基準が異なります。

✓
第1種有機溶剤：毒性が最も強い（クロロホルムなど）。屋内では密閉設備または局所排気装置の設置が義務
✓
第2種有機溶剤：最も種類が多い（トルエン・キシレン・酢酸エチルなど）。屋内では局所排気装置または全体換気装置が必要
✓
第3種有機溶剤：毒性が比較的低い（ガソリン・石油エーテルなど）。換気装置の設置は任意だが保護具の着用が必要
✓
有機溶剤作業主任者の選任・有機則健診（6ヶ月ごと）・作業環境測定（6ヶ月ごと）が義務

電離放射線障害防止規則（電離則）

✓
管理区域：外部放射線の実効線量が3ヶ月につき1.3mSvを超えるおそれのある区域に設定が必要
✓
放射線業務従事者の線量限度：実効線量は5年間で100mSv・かつ1年間で50mSvを超えてはならない
✓
放射線業務従事者には6ヶ月以内ごとに1回の健康診断実施が義務（記録は30年保存）
✓
エックス線装置を用いる作業・ガンマ線照射装置を用いる作業ではエックス線作業主任者またはガンマ線透過写真撮影作業主任者の選任が必要
有害業務関連に共通する「作業主任者」の選任義務
特化則・有機則・電離則・粉じん則など、有害業務関連の多くの規則で「作業主任者」の選任が義務付けられています。試験では「どの業務でどの作業主任者が必要か」という対応関係が問われます。特定化学物質作業主任者・有機溶剤作業主任者・エックス線作業主任者・特定粉じん作業主任者など、主要な作業主任者の種類と対応する業務を整理した一覧を作って覚えることをおすすめします。

8. 関係法令を効率よく攻略する3つの学習法

関係法令は覚える数値・規定が多い科目ですが、効率的なアプローチで学習すれば短期間で得点源にできます。合格者が実践している3つの学習法を紹介します。

学習法① 数値だけの一覧表を自分で作る

関係法令で出題される数値（人数・時間・頻度・保存期間）は限られています。すべての重要数値を1枚の表にまとめた「数値チートシート」を自分で作ることが最も効果的な学習法です。

具体的には「選任義務は50人以上」「産業医は50人以上・専属は1,000人以上」「衛生委員会は毎月1回」「定期健診は1年ごと・特定業務は6ヶ月ごと」「議事録は3年保存・健診記録は5年保存（一部30年）」といった数値を一覧化します。この表を通勤時間に見返すだけで、試験に出る数値の大半をカバーできます。

学習法② 過去問の「誤り選択肢」から学ぶ

関係法令の選択肢は、数値だけを変えた誤りパターンが非常に多く使われます。たとえば「衛生委員会の開催は2ヶ月に1回」（正しくは毎月1回）、「議事録の保存期間は5年」（正しくは3年）といった形で、数値を変えた誤り選択肢が繰り返し登場します。

過去問を解く際は「なぜこの選択肢は誤りなのか」を必ず確認し、正しい数値を上書きするように学習することで、出題者が仕掛ける数値変えの引っかけに対応できる力が身につきます。

学習法③ 第一種は有害業務関連を「別テーマ」として集中攻略する

第一種を受験する方は、有害業務以外の10問と有害業務関連の7問を別々のフェーズで学習することをおすすめします。まず第二種と共通の10問パートを完成させてから、有害業務関連の7問に集中投資するアプローチが効率的です。

有害業務関連は、特化則・有機則・電離則の3法令だけで例年4問以上出題されるため、この3法令を優先的に攻略するだけで大きな得点アップが期待できます。粉じん則・鉛則・高圧則は余裕があれば追加で押さえる程度で構いません。

関係法令 学習の重要ポイントまとめ
  • 数値（人数・頻度・保存期間）を一覧化した「数値チートシート」を自作する
  • 過去問の誤り選択肢のパターン（数値だけ変える）を把握して対策する
  • 第一種は有害業務以外10問を先に完成させ、その後有害業務関連7問に集中する
  • 特化則・有機則・電離則の3法令を最優先で攻略する
  • 作業主任者の種類と対応業務を対応表で整理する
  • 法改正情報を最新テキストで確認する（数値が変わっている場合あり）

9. まとめ

関係法令の出題傾向 まとめ

  • 関係法令は第一種17問・第二種10問と全科目中最多の出題数で合否を大きく左右する
  • 第二種は衛生管理体制・健康診断・労働時間の3テーマで5〜6問が出る——ここを押さえれば合格圏
  • 第一種固有の有害業務7問は特化則・有機則・電離則が最頻出——この3法令を優先する
  • 出題の中心は数値（人数・頻度・保存期間）の正確な記憶——数値チートシートを自作して活用する
  • 衛生管理者は50人以上・14日以内に選任・毎週1回巡視
  • 産業医は50人以上・1,000人超で専属・毎月1回巡視
  • 衛生委員会は50人以上・毎月1回開催・議事録3年保存
  • 健康診断の記録保存は原則5年（特化則第1類・電離放射線等は30年）
  • 過去問の誤り選択肢は「数値だけ変えた」パターンが多い——解説まで必ず確認する

関係法令は「覚えることが多くて難しい」という印象を持たれやすい科目ですが、出題されるポイントは意外と絞られています。毎年繰り返し出題されるテーマの数値を正確に覚え、過去問演習で問われ方のパターンを体得することが、関係法令攻略の王道です。当サイトの問題演習機能で、今日から得点を積み上げていきましょう。
