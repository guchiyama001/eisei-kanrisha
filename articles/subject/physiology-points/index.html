労働生理の頻出テーマ10選
【得点源にする攻略法】

衛生管理者試験の「労働生理」は、第一種・第二種ともに10問が出題される科目です。法律の暗記が中心の関係法令とは異なり、人体の仕組みを「理解」して解く問題が多く、正しく学習すれば得点源にしやすい科目でもあります。

この記事では、過去問の出題傾向を分析して厳選した頻出テーマ10項目を、試験に直結する重要ポイントとともに解説します。苦手意識がある方も、この記事を読み終えたころには「労働生理は得点源にできる」と感じていただけるはずです。

1. 労働生理はなぜ得点源になるのか

衛生管理者試験の3科目のうち、多くの受験者が最初に苦手意識を持つのが「労働生理」です。しかし実態は逆で、正しいアプローチで学べば3科目の中で最も安定して得点できる科目です。その理由を3点に整理します。

理由① 出題範囲がほぼ固定されている

労働生理の出題テーマは、毎年ほぼ同じ10〜12のカテゴリから出題されます。関係法令のように法改正で問われる数値が変わったり、労働衛生のように新しい制度が追加されたりするようなダイナミックな変化が少なく、学習した内容がそのまま試験に出やすいという特徴があります。

過去5年分の過去問を分析すると、血液・循環器、呼吸、神経系、感覚器、代謝・体温調節、疲労、睡眠の7テーマで毎年必ずといっていいほど出題されています。これらを重点的に押さえるだけで、10問中7〜8問は対応できます。

理由② 「理解」が「暗記」に変わると定着が速い

「心臓は4つの部屋に分かれている」「自律神経は交感神経と副交感神経からなる」——こうした知識は、単純な暗記ではなく「なぜそうなっているのか」という仕組みへの理解をセットにすると、記憶への定着が格段に速くなります。

たとえば「運動すると心拍数が増える」→「なぜなら筋肉への血流を増やすため」→「それを指令するのが交感神経」という流れで理解すると、関連する複数のテーマが芋づる式に記憶されます。これにより、表現が変わった問題にも対応できる応用力が生まれます。

理由③ 問題の選択肢が「正誤判定しやすい」

労働生理の問題は、選択肢の正誤が比較的明確です。「血液は酸素を運ぶ」「肺で二酸化炭素を排出する」といった基本的な機能の正誤を問うものが多く、知識があれば確実に正答できます。関係法令のように「50人以上か100人以上か」といった数値の紛らわしさが少ないため、知識の精度より理解の深さが得点に直結します。

労働生理の試験上の位置づけ
第一種・第二種ともに労働生理は10問で構成され、合格には各科目40%以上（10問中4問以上）の正答が必要です。ただし関係法令や労働衛生は問題数が多い（第一種で各17問）ため、労働生理は問題数が少ない分だけ1問の重みが大きく、苦手なままにしておくと足切りリスクが高まります。10問中6〜7問の正答を目標として学習することを推奨します。

以降では、頻出テーマを1つずつ、試験に出るポイントを絞って解説していきます。

2. 頻出テーマ① 血液・循環器系（心臓・血管・血液の成分）

出題頻度：毎年必出

血液・循環器系は、労働生理の中で最も出題頻度が高いテーマです。「血液の成分と役割」「心臓の構造と血液循環」「血圧とその調節」の3つのサブテーマを中心に出題されます。

血液の成分と役割

血液は大きく血漿（けっしょう）と血球に分けられます。血漿は血液の約55%を占める液体成分で、水分・タンパク質・ホルモン・栄養素などを含み、物質の運搬と体温調節に関わります。

血球には赤血球・白血球・血小板の3種類があります。それぞれの役割は試験に頻繁に出るため、確実に整理しておきましょう。

血球の種類主な役割試験で問われやすいポイント
赤血球 ヘモグロビンが酸素を運搬 核がない・寿命約120日・骨髄で生成
白血球 免疫・異物の排除（食作用） 核がある・感染時に増加・好中球・リンパ球などに分類
血小板 止血（血液凝固） 核がない・最も小さい・フィブリンを形成

心臓の構造と血液循環

心臓は右心房・右心室・左心房・左心室の4つの部屋で構成されています。体から戻ってきた静脈血は右心房→右心室→肺（肺循環）で酸素を受け取り、動脈血となって左心房→左心室→全身（体循環）へ送り出されます。

✓
肺動脈には静脈血が流れる（酸素が少ない血液）——名前と血液の種類が逆になる点が頻出
✓
肺静脈には動脈血が流れる（酸素が多い血液）——同上
✓
心臓の収縮によって血液を送り出す際の圧力を収縮期血圧（最高血圧）、弛緩時を拡張期血圧（最低血圧）という
✓
心拍数の増加・血管収縮→交感神経が支配。心拍数の減少・血管拡張→副交感神経が支配
「動脈＝動脈血」は誤り——よく出る引っかけ
「動脈には動脈血が流れる」と思いがちですが、肺動脈には酸素の少ない静脈血が流れます。「動脈・静脈」は血管の種類（心臓から出る・心臓に戻る）を指し、「動脈血・静脈血」は酸素量（多い・少ない）を指します。この違いを明確に理解しておくことが重要です。

3. 頻出テーマ② 呼吸器系（呼吸のしくみ・肺機能）

出題頻度：毎年必出

呼吸器系は「呼吸のしくみ」「ガス交換」「呼吸に関わる筋肉と神経」を中心に出題されます。特に外呼吸・内呼吸の違いと、呼吸を調節する仕組みは毎年のように問われます。

外呼吸と内呼吸の違い

呼吸には外呼吸と内呼吸の2種類があります。外呼吸とは肺胞と毛細血管の間で行われるガス交換のことで、空気中の酸素を血液に取り込み、血液中の二酸化炭素を排出します。一方、内呼吸とは血液と全身の細胞・組織の間で行われるガス交換のことで、組織に酸素を渡し、代謝で生じた二酸化炭素を受け取ります。

種類場所交換内容
外呼吸 肺胞 ↔ 毛細血管 空気から酸素を取り込み、CO₂を排出
内呼吸 毛細血管 ↔ 組織・細胞 組織に酸素を渡し、CO₂を受け取る

呼吸運動のしくみ

呼吸運動は横隔膜（おうかくまく）と肋間筋（ろっかんきん）によって行われます。吸気（息を吸う）時には横隔膜が下がり肋間筋が収縮して胸腔が広がり、肺が膨らみます。呼気（息を吐く）時にはこれが逆になります。

✓
呼吸の調節中枢は延髄にある。呼吸リズムは延髄の呼吸中枢が自動的に制御する
✓
血液中のCO₂濃度上昇が呼吸を促進する最大の刺激。O₂濃度の低下も刺激になるが、CO₂が主役
✓
成人の安静時呼吸数は約15〜20回/分
✓
肺自体には筋肉がなく、胸腔の容積変化に受動的に従って膨縮する
呼吸量に関する数値も押さえておこう
  • 1回換気量（安静時）：約500mL
  • 予備吸気量：約1,500mL（さらに吸える量）
  • 予備呼気量：約1,500mL（さらに吐ける量）
  • 肺活量：約3,500〜4,000mL（最大に吸って最大に吐いた量）
  • 残気量：約1,500mL（最大に吐いても肺に残る量）
これらの数値そのものが問われることは少ないですが、「残気量は肺活量に含まれない」といった概念の理解は頻出です。

4. 頻出テーマ③ 消化器系・肝臓の役割

出題頻度：高め

消化器系では「消化の種類」「消化液と消化酵素」「肝臓の多彩な機能」が頻出です。特に肝臓の役割は毎年のように問われ、複数の機能を正確に覚えることが求められます。

消化の種類と消化液

消化には機械的消化（咀嚼・蠕動（ぜんどう）運動など）と化学的消化（消化酵素による分解）があります。化学的消化の中心は消化酵素で、どの消化液にどの酵素が含まれるかが試験に出ます。

消化液分泌器官主な消化酵素分解対象
唾液唾液腺アミラーゼ（プチアリン）デンプン（糖質）
胃液胃ペプシンタンパク質
膵液膵臓アミラーゼ・トリプシン・リパーゼ糖質・タンパク・脂質（3大栄養素すべて）
胆汁肝臓（胆嚢に貯蔵）消化酵素を含まない脂肪の乳化（吸収を助ける）
腸液小腸マルターゼなど糖質の最終分解

肝臓の役割

肝臓は人体最大の臓器であり、500以上の化学反応が行われる「生体の化学工場」とも呼ばれます。試験では肝臓の機能を問う問題が毎年出題されます。以下の機能を確実に押さえましょう。

✓
糖質代謝：グルコースをグリコーゲンとして貯蔵し、血糖が下がると分解して放出する
✓
タンパク質合成：血漿タンパク（アルブミン・フィブリノーゲンなど）を合成する
✓
解毒作用：有害物質（アルコール・薬物・アンモニアなど）を無害化する
✓
胆汁の生成：脂肪の消化を助ける胆汁を生成し、胆嚢に貯蔵する
✓
熱産生：代謝活動が活発なため、体温維持に貢献する体内最大の熱産生臓器
「胆汁に消化酵素は含まれない」は頻出の引っかけ
胆汁は脂肪を乳化させて消化を助けますが、それ自体に消化酵素は含まれません。「胆汁のはたらきは脂肪を乳化すること」と明確に覚えておきましょう。胆汁を生成するのは「肝臓」で、貯蔵するのは「胆嚢（たんのう）」です。この2つの区別も問われます。

5. 頻出テーマ④ 腎臓・泌尿器系（尿生成のしくみ）

出題頻度：高め

腎臓・泌尿器系では「尿の生成過程」「腎臓の構造」「尿に含まれる・含まれない成分」が主な出題テーマです。特に「糸球体」「尿細管」での濾過と再吸収のプロセスは頻出です。

尿の生成プロセス

腎臓では毎分約1リットルもの血液が流れ込み、老廃物が濾過されて尿が生成されます。このプロセスは2段階で行われます。

第1段階は濾過です。腎臓の基本単位であるネフロン（腎単位）の糸球体（しきゅうたい）で、血液中の水分・老廃物・電解質などが「原尿」として濾過されます。1日に生成される原尿量は約150〜180リットルと非常に多量です。

第2段階は再吸収です。原尿は尿細管（にょうさいかん）を通過する過程で、体に必要な水分・グルコース・電解質などが毛細血管へ再吸収されます。最終的に尿として排出されるのは約1〜1.5リットルで、原尿の約99%が再吸収されることになります。

✓
健康な人の尿にはグルコース（糖）・タンパク質は含まれない（腎臓で再吸収される）
✓
尿には尿素・クレアチニン・尿酸・無機塩類などの老廃物が含まれる
✓
腎臓はエリスロポエチン（赤血球の生成を促すホルモン）も分泌する
✓
腎臓はレニンを分泌し、血圧調節にも関わる
腎臓の構造：皮質と髄質
腎臓は外側の皮質と内側の髄質に分けられます。糸球体とボーマン嚢（のう）は皮質に存在し、尿細管の一部は髄質に伸びています。試験では「糸球体は皮質にある」という事実が問われることがあります。また、腎臓は左右に1個ずつあり、1つの腎臓に約100万個のネフロンが存在します。

6. 頻出テーマ⑤ 神経系（自律神経・中枢・末梢）

出題頻度：毎年必出

神経系は労働生理の中でも最重要テーマの一つです。「中枢神経と末梢神経の区別」「自律神経の2種類とその働き」「脳の各部位の役割」が主な出題内容で、毎年1〜2問出題されます。

神経系の分類

神経系は大きく中枢神経系（脳・脊髄）と末梢神経系に分類されます。末梢神経系はさらに体性神経系（随意運動・感覚）と自律神経系（不随意の臓器コントロール）に分けられます。

自律神経系：交感神経と副交感神経

自律神経系は交感神経と副交感神経の2つからなり、多くの臓器はこの2系統による二重支配を受けています。試験では両者の違いを問う問題が毎年出ます。

器官・機能交感神経（緊張・活動時）副交感神経（安静・回復時）
心拍数増加減少
血圧上昇低下
気管支拡張収縮
消化管運動抑制（蠕動低下）促進（蠕動亢進）
瞳孔散大縮小
唾液分泌少量・粘稠多量・漿液性
汗腺発汗促進（ほぼ支配しない）

脳の各部位の役割

脳は大脳・小脳・脳幹（間脳・中脳・橋・延髄）に分けられ、それぞれ異なる機能を担います。

✓
大脳：高次機能（思考・判断・記憶・言語・随意運動）を担う。大脳皮質と大脳辺縁系に分かれる
✓
小脳：運動の協調・平衡感覚・姿勢の維持を担う
✓
間脳（視床下部）：自律神経の最高中枢。体温調節・水分バランス・食欲・内分泌調節を担う
✓
延髄：呼吸・循環・嚥下などの生命維持に直結する反射中枢
「自律神経の最高中枢は視床下部」は最頻出
  • 自律神経の最高中枢＝視床下部（間脳）
  • 呼吸の調節中枢＝延髄
  • 体温調節の中枢＝視床下部
  • 平衡感覚・運動協調＝小脳
各部位と機能の対応は確実に覚えておきましょう。「延髄が呼吸中枢」は特に頻出です。

7. 頻出テーマ⑥ 感覚器（視覚・聴覚・その他）

出題頻度：高め

感覚器では「視覚の仕組み」「聴覚の仕組み」「皮膚感覚」が主な出題テーマです。目と耳の構造については、特定の部位の名称と機能が問われます。

視覚の仕組み

眼球は外から光を取り込み、網膜に像を結ぶ光学装置です。試験では各部位の名称と機能を正確に理解することが求められます。

✓
角膜・水晶体：光を屈折・収束させ、網膜に像を投影する。水晶体は毛様体筋の収縮で厚みが変わり焦点を調節（遠近調節）
✓
網膜：視細胞（錐体・桿体）が存在し、光を神経信号に変換する
✓
錐体（すいたい）：明所・色の識別を担う。中心窩に多く集まる
✓
桿体（かんたい）：暗所・明暗の識別を担う。周辺部に多い。暗順応に関わる
✓
瞳孔：虹彩の中央部で光量を調節。明るいと縮小（縮瞳）、暗いと拡大（散瞳）

聴覚・平衡感覚の仕組み

耳は外耳・中耳・内耳の3部位に分けられます。音の伝達経路と平衡感覚の器官については、試験で出題されやすい知識をまとめます。

✓
音の伝達経路：外耳→鼓膜（中耳）→耳小骨→内耳（蝸牛）→聴神経→大脳
✓
半規管（内耳）：回転感覚・加速度を感知。平衡感覚に関与
✓
耳石器（前庭）：直線加速度・重力を感知。乗り物酔いに関連
✓
騒音による聴力損失は4,000Hz（4kHz）付近から始まる——騒音性難聴の特徴
皮膚感覚の種類
皮膚感覚には触覚・圧覚・温覚・冷覚・痛覚の5種類があります。これらは別々の感覚受容器が担います。「皮膚には温覚と冷覚の両方がある」「痛覚受容器は全身に分布する」といった点が問われることがあります。なお、皮膚の感覚点の密度は部位によって異なり、指先は特に高密度です。

8. 頻出テーマ⑦ 筋肉・骨格系（随意筋・不随意筋・疲労）

出題頻度：中程度

筋肉・骨格系では「筋肉の種類」「筋収縮のエネルギー源」「筋疲労のメカニズム」が主な出題テーマです。特に「随意筋」「不随意筋」「平滑筋」「心筋」の分類は正確に覚える必要があります。

筋肉の種類と分類

種類場所随意・不随意形態
骨格筋 四肢・体幹などの骨に付着 随意筋（意志で動かせる） 横紋筋（縞模様がある）
心筋 心臓 不随意筋（意志で動かせない） 横紋筋（縞模様がある）
平滑筋 内臓・血管壁など 不随意筋（意志で動かせない） 平滑筋（縞模様がない）
「心筋は横紋筋・不随意筋」——この組み合わせが頻出の引っかけ
「横紋筋＝随意筋」と思いがちですが、心筋は横紋筋でありながら不随意筋です。この例外を意識して覚えておくことが重要です。試験では「心臓の筋肉は不随意筋か随意筋か」「心筋は横紋筋か平滑筋か」という問い方でよく出ます。

筋収縮のエネルギーと疲労

筋肉が収縮するためにはエネルギーが必要です。その即時的エネルギー源はATP（アデノシン三リン酸）で、ATPは主にグルコース（糖質）の酸化分解によって再生されます。

✓
酸素が十分にある場合：グルコース→ピルビン酸→CO₂＋H₂O（好気的代謝）
✓
酸素が不足する場合：グルコース→乳酸（嫌気的代謝）——筋疲労の一因
✓
筋疲労の主原因は乳酸の蓄積ではなく、エネルギー（ATP）の枯渇と代謝産物の蓄積とされる（近年の見解）
✓
筋肉はグリコーゲンを貯蔵しており、必要に応じてグルコースに分解してエネルギー源とする

9. 頻出テーマ⑧ 代謝・体温調節（基礎代謝・BMR・体温）

出題頻度：高め

代謝・体温調節では「基礎代謝の定義」「体温調節のしくみ」「熱産生と放熱の方法」が頻出です。特に基礎代謝の定義と条件は毎年のように問われます。

基礎代謝とは

基礎代謝（BMR：Basal Metabolic Rate）とは、生命を維持するために必要な最小限のエネルギー代謝量のことです。具体的には「早朝空腹時・安静・快適な室温」という条件下で測定されます。

✓
基礎代謝量は成人男性で約1,500kcal/日、成人女性で約1,200kcal/日が目安
✓
基礎代謝量は青年期（15〜17歳頃）に最大となり、その後加齢とともに低下する
✓
同じ体重でも筋肉量が多い人ほど基礎代謝量が高い
✓
甲状腺ホルモンは基礎代謝を促進する。甲状腺機能亢進症では代謝が異常に上がる

体温調節のしくみ

人体は体温を約36〜37℃に維持するため、常に熱産生と放熱のバランスをとっています。体温調節の中枢は視床下部です。

熱産生（体温を上げる方向）放熱（体温を下げる方向）
筋肉のふるえ（シバリング） 発汗（蒸発）
代謝の促進（甲状腺ホルモン等） 皮膚血管の拡張（輻射・対流）
皮膚血管の収縮（放熱を防ぐ） 呼気からの水分蒸発
体温調節に関する試験頻出ポイント
  • 発汗は蒸発熱によって熱を放散する最も効果的な放熱手段
  • 高温環境での発汗は大量の水分・電解質（特にNaCl）を失う——熱中症と関連
  • 体温調節中枢＝視床下部（間脳）——神経系でも同様に問われる
  • 体温は直腸温＞口腔温＞腋窩（えきか）温の順に高い
  • 女性の体温は排卵後（黄体期）に上昇する——基礎体温と関連

10. 頻出テーマ⑨ 疲労・ストレス（産業疲労・メンタル）

出題頻度：高め

疲労・ストレスは「産業疲労の分類」「疲労の評価方法」「ストレスの生体反応」が主な出題テーマです。職場の健康管理と直結するテーマであり、近年の試験では精神的疲労・ストレスに関する問題も増えています。

産業疲労の分類

産業疲労は発生原因や現れ方によっていくつかの視点から分類されます。

分類の視点種類内容
原因による分類 身体的疲労 筋肉・身体器官の疲労。重労働・長時間立ち仕事など
精神的疲労 精神的緊張・集中・感情労働によるもの。VDT作業・接客業など
期間による分類 急性疲労 1日の作業後に生じる疲労。休息で回復できる
慢性疲労（蓄積疲労） 回復が追いつかず疲労が蓄積した状態。過労死との関連

疲労の評価方法

疲労の程度を客観的に評価する方法としては、以下のようなものが用いられます。

✓
フリッカー検査（ちらつき検査）：光のちらつきが「定常光」に見える周波数（CFF値）を測定。疲労するとCFF値が低下する。精神的疲労の評価に有効
✓
連続加算作業（クレペリン検査）：計算作業の速度や正確性から作業能力の変化を測定する
✓
握力検査：筋力の低下から身体的疲労を測定する

ストレスの生体反応

ストレスとは、外部からの刺激（ストレッサー）に対する心身の反応です。ストレス反応は自律神経系・内分泌系（ホルモン）を通じて全身に影響します。

✓
急性ストレス応答：交感神経が優位になり、アドレナリン・ノルアドレナリンが分泌され、心拍数増加・血圧上昇・血糖上昇が生じる（「闘争か逃走」反応）
✓
慢性ストレスでは副腎皮質からコルチゾール（糖質コルチコイド）が持続的に分泌され、免疫機能の低下・消化器疾患・うつ病などのリスクが高まる
✓
セリエの汎適応症候群：ストレスへの生体反応は「警告反応期→抵抗期→疲弊期」の3段階をたどる
産業疲労と過重労働
慢性的な疲労の蓄積は過重労働につながり、脳・心臓疾患（いわゆる過労死）のリスクを高めます。厚生労働省の過労死等防止対策では、時間外労働が月80時間超を「過重労働」の目安とし、産業医による面接指導の対象としています。こうした数値は関係法令との連動で出題される場合もあるため、セットで理解しておくと効果的です。

11. 頻出テーマ⑩ 睡眠・生体リズム（概日リズム・睡眠段階）

出題頻度：高め

睡眠・生体リズムは「概日リズム（サーカディアンリズム）」「睡眠の段階（レム睡眠・ノンレム睡眠）」「睡眠と健康の関係」が出題テーマです。近年、交代制勤務や長時間労働との関連で注目されているテーマです。

概日リズム（サーカディアンリズム）

人体には約24時間周期の生体リズム（概日リズム）が存在し、睡眠・覚醒・体温・ホルモン分泌などがこのリズムに従って変動します。この周期のコントロール中枢は視床下部の視交叉上核（しこうさじょうかく）です。

✓
概日リズムは光によってリセットされる。朝の光を浴びることで体内時計が調整される
✓
メラトニン（松果体から分泌）：暗くなると分泌増加し眠気を促す。概日リズムの調節に関与
✓
深部体温は夕方〜夜に最高値→就寝前に低下するパターンをとる。体温低下が睡眠を誘発する
✓
交代制勤務者は概日リズムの乱れが生じやすく、消化器疾患・心血管疾患・がんリスクの上昇が報告されている

レム睡眠とノンレム睡眠

睡眠は大きくレム睡眠（REM睡眠）とノンレム睡眠（NREM睡眠）に分けられ、一晩の睡眠中に約90分周期でこの2種類が交互に現れます。

レム睡眠ノンレム睡眠
英語の意味Rapid Eye Movement（急速眼球運動）あり眼球運動なし
脳波の特徴覚醒に近いパターン徐波（深い睡眠）
夢見やすいほとんど見ない
筋肉の状態脱力（体は休んでいる）筋緊張は残る
記憶の整理手続き記憶の定着陳述記憶の定着
成長ホルモンほぼ分泌なし深い段階で大量分泌
睡眠に関する試験頻出ポイント
  • 成長ホルモンはノンレム睡眠（特に深い段階）に多く分泌される
  • 夢を見やすいのはレム睡眠の時
  • 睡眠サイクルは約90分周期でレム・ノンレムが繰り返される
  • 加齢とともにレム睡眠の割合は増え、深いノンレム睡眠（徐波睡眠）は減少する
  • 睡眠不足はコルチゾール分泌増加・食欲増進（グレリン増加）・免疫低下をもたらす

12. 労働生理を効率よく攻略する3つの学習法

10の頻出テーマを解説してきました。最後に、これらの知識を効率よく得点に結びつけるための学習法を3つお伝えします。

学習法① 「つながり」を意識した理解型学習

労働生理の各テーマは互いに深くつながっています。たとえば「交感神経が優位になると心拍数が増加し血糖が上がり瞳孔が散大する」という連鎖を1つの流れとして理解することで、バラバラな暗記よりもはるかに記憶が定着します。

具体的には、学習中に「なぜそうなるのか」という疑問を意識的に持つことが重要です。「汗をかくのはなぜ？」→「体温を下げるため」→「発汗は蒸発熱を利用する」→「体温調節中枢は視床下部」という芋づる式の理解が、本番での応用力につながります。

学習法② 過去問で「問われ方のパターン」を体得する

労働生理では、同じ知識が表現を変えて繰り返し出題されます。「肺動脈に流れる血液の種類」「心筋が随意筋か不随意筋か」「成長ホルモンが分泌される睡眠段階」——これらは毎年のように出題される定番問題です。

過去問を解く際には、正誤を確認するだけでなく「なぜ他の選択肢が誤りなのか」を解説で確認することが重要です。誤りの選択肢のパターンを理解することで、初見の問題にも対応できる力が育ちます。当サイトでは年度別・科目別に過去問を演習できるため、ぜひ活用してください。

学習法③ 図や表で視覚的に整理する

神経系・循環器系・消化器系など、体の構造に関する知識は図や表で整理すると定着が早まります。特に自律神経の交感神経・副交感神経の比較表、血液循環の経路図、睡眠段階のグラフなどは、自分でノートに書き出すことで記憶が強化されます。

完璧な図を描く必要はなく、「交感神経＝緊張・活動」「副交感神経＝安静・回復」といったキーワードと具体的な体の変化をセットにした箇条書きで十分です。スキマ時間に見返せるように、スマートフォンで写真を撮って持ち歩くのも有効な活用法です。

労働生理 頻出テーマ 重要度まとめ
  • 毎年必出（最優先）：血液・循環器系 ／ 呼吸器系 ／ 神経系（自律神経）
  • 高頻度（次に優先）：消化器・肝臓 ／ 腎臓 ／ 感覚器 ／ 代謝・体温調節 ／ 疲労・ストレス ／ 睡眠
  • 中頻度（余裕があれば）：筋肉・骨格系 ／ 内分泌系 ／ 免疫系
学習時間が限られている場合は、まず「毎年必出」の3テーマを完成させることを最優先にしてください。この3テーマだけで10問中3〜4問は確保できます。

まとめ

労働生理 頻出テーマ10選 まとめ

  • ① 血液・循環器系：赤血球の役割・肺動脈には静脈血が流れる・自律神経と心拍の関係
  • ② 呼吸器系：外呼吸・内呼吸の違い・CO₂が呼吸促進の主要刺激・延髄が呼吸中枢
  • ③ 消化器・肝臓：胆汁に消化酵素なし・肝臓の多彩な機能（解毒・合成・貯蔵）
  • ④ 腎臓・泌尿器：糸球体で濾過→尿細管で再吸収・尿にグルコースは含まれない
  • ⑤ 神経系：交感・副交感の対比・視床下部が自律神経最高中枢・延髄が呼吸中枢
  • ⑥ 感覚器：錐体（明所・色）と桿体（暗所・明暗）・騒音性難聴は4kHz付近から
  • ⑦ 筋肉・骨格系：心筋は横紋筋かつ不随意筋・乳酸と筋疲労の関係
  • ⑧ 代謝・体温調節：基礎代謝は青年期に最大・体温調節中枢は視床下部
  • ⑨ 疲労・ストレス：フリッカー検査・蓄積疲労・コルチゾールと慢性ストレス
  • ⑩ 睡眠・生体リズム：成長ホルモンはノンレム睡眠に分泌・メラトニンと概日リズム

労働生理は「暗記する科目」ではなく「理解する科目」です。人体の仕組みをひとつひとつ丁寧に理解していくことで、問われ方が変わっても対応できる本物の知識が身につきます。この記事で解説した10テーマを軸に、過去問演習を重ねて得点源に育てていきましょう。
